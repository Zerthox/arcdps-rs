Expand description

Subscribes to unofficial extras callbacks after checking for compatibility.

Unsupported callbacks will be skipped.

§Safety

Info needs to point to a valid to interpret as subscriber infos with minimum version of ExtrasVersion::MIN_SUB_INFO. It is passed as pointer to prevent UB by creating a reference to the wrong type.

Name needs to be null-terminated.
