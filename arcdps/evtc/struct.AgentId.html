Expand description

Ids for an agent.

Fields§

§id: u64

Agent id as assigned by Arc.

§instance_id: u16

Instance id of the agent as appears in game at time of event.

§master_instance_id: u16

If agent has a master (e.g. is minion), will be equal to instance id of master, zero otherwise.
