pub struct Log {
    pub header: Header,
    pub agents: Vec<Agent>,
    pub skills: Vec<Skill>,
    pub events: Vec<Event>,
}
Expand description

An EVTC log.

Fields§

§header: Header

The log header with meta information.

§agents: Vec<Agent>

Agents (entities) present in the log.

§skills: Vec<Skill>

Information about skills used in the log.

§events: Vec<Event>

Every Event occurring in the log.

Some events may also hold meta information, for example StateChange::BuffFormula.
